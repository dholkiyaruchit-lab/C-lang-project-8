🧩 Programs Included

<h4>🔹 Q1: String Length Using Pointer </h4>

onlineGDB : https://onlinegdb.com/U3HbOXaXD

This program calculates the length of a string without using any built-in functions.

💡 Concept Used:
Pointers
String traversal
While loop

⚙️ How It Works:
User inputs a string
A pointer (ptr) points to the string
Loop runs until '\0' (null character)
Length is counted manually

▶️ Sample Output:
Enter any string: Hello World
The length of a string is: 11

<h4>🔹 Q2: Cube of 2D Array Elements Using Pointer </h4>

onlineGDB : https://onlinegdb.com/Ix7pfwLCe

This program calculates the cube of each element in a 2D array using pointers.

💡 Concept Used:
Pointers with arrays
2D array handling
Pointer arithmetic
Functions

⚙️ How It Works:
User inputs size of square matrix
Elements are stored in a 2D array
Array is passed as pointer (&arr[0][0])
Each element is cubed using pointer arithmetic

▶️ Sample Output:
Enter array's size: 2

Enter array elements:
a[0][0] = 2
a[0][1] = 3
a[1][0] = 4
a[1][1] = 5

Cubes of all elements:
8 27
64 125

📚 Learning Outcomes
Understanding pointers in C
Working with strings without built-in functions
Handling 2D arrays using pointers
Function parameter passing using pointers
